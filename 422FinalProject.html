Dateset Description: Student Grades¶

This dataset portrays the results of a survey conducted of students taking both math and portuguese language courses in secondary school in Portugal. The survey contains information regarding the background of the student and their family life, their academic history, as well as their social life and activities. Some important variables in this dataset that can be analyzed in response to this background information are 'G1', 'G2', and 'G3'. These features refer to period one, period two, and final academic grades, where each period refers to an academic half-year. The dataset contains a total of 395 entries, meaning that 395 surveys of students were conducted as part of the data collection. (words: 111)

Regression Models

¶

From our initial hypothesis and the articles we had read, our assumption was that alcohol does affect exam grades. Hence we decided to see if we could build a regression model to see this correlation. We built 2 regression models to see how different features in our dataset affected grades.

First Experiment¶

Initially, we took the parameters "faliures", "studytime", "absences", "G2" to get a basic understanding to see if purely academic parameters such as past failiures, studytime, etc affected final grades.

As we can see, the accuracy we got with this was 78.84% which is very good. And shows that academic factors have a strong corellation with grades.

Second Experiment¶

Our assumption was that we should technically see the inverse of this while considering Alcohol's affect on grades (i.e. Less alcohol conspution should technically imply higher grades) - but the results were surprising. We found that there was no correlation. On building the linear regression models with the features "goout", "Walc", "Dalc", and "studytime" vs "G3", we can see that the accuracy is 3.15% which is extremely low. From the scatter plot we can also notice that there is no specific trend we get from this.

Methodology¶

  • Extracted the features I needed
  • Used sklearn's test_train_split method to split the dataset up into test and train data.
  • Fit the model to a linear regression curve on the train data.
  • Calculate the accuracy against the test data.
In [ ]:
# Create a dataframe with only the columns we want to use
df2 = df[['failures', 'studytime', 'absences', 'G2']]
df2.head()

x = df2
y = df['G3']

# Split the data into train and test sets
x_train, x_test, y_train, y_test = train_test_split(x, y, test_size=0.2, random_state=15)

model = LinearRegression()

model.fit(x_train, y_train)

y_pred = model.predict(x_test)

score = model.score(x_test, y_test)
print("Accuracy: ", score)

# Graph test and train data to check for overfitting
plt.scatter(y_test, y_pred)
plt.xlabel('True Values')
plt.ylabel('Predictions')

# Graph linear regression line
plt.plot([y_test.min(), y_test.max()], [y_test.min(), y_test.max()], '-', lw=4)

plt.title('Linear Regression Model built')

Concluding Remarks

¶

We found that building a model using features such as failures', 'studytime', 'absences', and 'G2' yielded much better a accuracy that the model built using 'goout', 'studytime', 'Walc', and 'Dalc'. The first model yielded an accuracy of approximately 78.8% while the second model had an accuracy of only 3.2% This is in line with what we expected based on our preliminary analysis and the second model is included in this report to serve as a contrast with the first model, essentially to show how poorly a model can perform if built on the wrong features.
